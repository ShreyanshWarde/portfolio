# Shreyansh Warde — Portfolio

Personal portfolio website built with React + TypeScript + Vite.

## Deploy to Vercel (3 steps)

### Step 1 — Push to GitHub
```bash
git init
git add .
git commit -m "initial portfolio"
# Create a new repo on github.com, then:
git remote add origin https://github.com/ShreyanshWarde/portfolio.git
git push -u origin main
```

### Step 2 — Import on Vercel
1. Go to [vercel.com](https://vercel.com) → **Add New Project**
2. Import your GitHub repo
3. Vercel auto-detects Vite — keep all defaults
4. Click **Deploy**

### Step 3 — Done
Your site is live at `https://your-project.vercel.app`

Optional: Add a custom domain in Vercel → Project → Settings → Domains.

## Local dev
```bash
npm install
npm run dev
```
